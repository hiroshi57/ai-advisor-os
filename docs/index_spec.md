# AI-Adoption Index 仕様書 (draft v0.1)

> ステータス: **仕様書ドラフト（承認待ち）** / 対象: B2 ai-advisor-os / 作成: 2026-07-09

## 0. 目的

各クライアントのAI活用度を独自スコア **AI-Adoption Index（0–100）** で毎月算出し、
業界横断の匿名ベンチマークと合わせて「同業界での位置」を示す（単独コンサルには作れない構造的差別化）。

## 1. 算出式（4因子の重み付き合成）

```
index = w_util*利用率 + w_breadth*活用の幅 + w_depth*深さ + w_gov*ガバナンス
初期重み(formula.yaml): w_util=0.35, w_breadth=0.2, w_depth=0.25, w_gov=0.2
```

| 因子 | 定義 | 0–100 の算出 |
|------|------|-------------|
| 利用率 utilization | WAU/対象者（申告 or ツールAPI実測） | 割合×100 |
| 活用の幅 breadth | 利用部署数 / 全部署数 | 割合×100 |
| 深さ depth | 用途段階(定型文=1 / 分析=2 / 意思決定支援=3)の加重平均 | (平均段階/3)×100 |
| ガバナンス governance | ルール整備チェックリスト充足率 | 充足項目/全項目×100 |

- `formula.yaml` にバージョン番号を持たせ、スコアにひも付ける（経時比較性の担保）。

## 2. 匿名ベンチマーク（k-匿名性）

- 業界別・規模別に全クライアント横断で中央値・四分位を集計。
- **同一セグメントの企業数が5社未満の場合はベンチマーク非表示**（k=5 の k-匿名性）。
- レポート表示: 「あなたは同業界で上位◯%」。

## 3. 受け入れ基準

- 5社未満セグメントでベンチマークが非表示になること（テスト担保）。
- `formula.yaml` のバージョンを変えても、過去スコアの再計算と旧版参照が両立すること。
